# DIAGRAMA ENTIDADE-RELACIONAMENTO (DER)

## SOBRE:

O Diagrama Entidade-Relacionamento é uma representação visual da estrutura do banco de dados. Ele mostra as entidades existentes no sistema, seus atributos e os relacionamentos entre elas.

Esse diagrama auxilia no planejamento e na compreensão da base de dados, permitindo visualizar como as informações se conectam antes da implementação do banco de dados.

### Componentes do DER

**Entidade:** Representa um objeto ou conceito do sistema sobre o qual informações são armazenadas.

**Atributo:** Característica que descreve uma entidade.

**Relacionamento:** Associação existente entre duas ou mais entidades.

**Cardinalidade:** Define a quantidade de ocorrências que podem participar de um relacionamento (1:1, 1:N ou N:N).

**Chave Primária:** Atributo responsável por identificar unicamente cada entidade.

**Chave Estrangeira:** Atributo utilizado para estabelecer ligações entre entidades relacionadas.
