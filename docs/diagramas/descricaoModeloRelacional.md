# DESCRIÇÃO DO MODELO RELACIONAL

## SOBRE:

O modelo relacional apresenta a organização dos dados do sistema no banco de dados. Ele é composto por tabelas que armazenam as informações da aplicação e por relacionamentos que conectam essas tabelas.

Por meio desse modelo é possível visualizar como os dados estão estruturados, quais informações cada tabela armazena e como ocorre a comunicação entre elas. Essa representação também auxilia na identificação de chaves primárias e estrangeiras, garantindo a integridade e a consistência dos dados.
